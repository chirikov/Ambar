	Установка
	
Итак, у Вас на компьютере должны быть установлены :
- Microsoft Windows XP(Home Edition/Professional)*
- HTTP сервер (с программой поставляется Apache 2.0.48);
- PHP интерпритатор (с программой поставляется PHP версии 4.3.4);
- Сервер MySQL (с программой поставляется MySQL версии 3.23.53);
- Интернет-браузер (желательно использовать поставляющийся с программой Internet Explorer 6);

*Примечание: необходимость в использовании Microsoft Windows XP отпадает, если Вы будете
использовать свои компоненты (сервер, РНР ...), не поставляющиеся с программой и сумеете
их правильно установить и настроить.

Внимание!!! Версия 2.9 не приспособлена для работы в Unix-системах, где требуется выставлять
права для файлов.

За инструкцией по установке сервера и других компонент, прилагающихся к программе,
обращайтесь к файлу "install_server.txt", который находится на компакт-диске.

Вы должны знать имя сервера MySQL, имя пользователя и пароль для доступа к серверу
MySQL. Вы должны еще знать, в какой папке на сервере находятся файлы программы
(то есть в какую папку Вы производили установку программы), и URL-адрес этой папки
(то есть по какому адресу, вводимому в адресную строку браузера, доступна эта папка).

Теперь, когда у Вас есть установленный и настроенный сервер с обработчиком PHP-кода
и сервером MySQL, Вы можете приступить ко вторичной установке программы.
Для этого откройте окно интернет-браузера и в адресной строке введите адрес файла
"install.php" (например, "http://localhost/ambar/install.php"). Следуйте
инструкциям на экране. Если Вы увидите надпись "Поздравляю! ..." без сообщений об
ошибках, значит программа установлена правильно.

Теперь в папке, адрес которой Вы указали при вторичной установке
(например, "http://localhost/ambar") есть ярлык, который служит для
запуска программы.

Внимание! Для запуска программы следует использовать только этот ярлык или его копии!
	  Никогда не обновляйте страницы программы (сочетание клавиш "Ctrl + R")!!!

Пароль по умолчанию - password
Смените пароль сразу после полной установки программы! ("Изменение настроек" -> "Изменить пароль")

--------------------------------------------------------------------------------------------------
	Удаление программы.

Чтобы удалить программу, используйте "Панель управления" -> "Установка и удаление программ"
(для Microsoft Windows).
Чтобы удалить сервер и все его компоненты, удалите HTTP-сервер и сервер MySQL через
"Панель управления" -> "Установка и удаление программ" (для Microsoft Windows).
